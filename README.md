## Featured work

### MMA-TP — Modular Multi-State AI Teaching Protocol

Original creator and corresponding author of **MMA-TP**, a published protocol-level framework for structuring multi-turn LLM tutoring interactions through paired system prompts and JSON-based state specifications. The end product is a copy-and-paste AI tutor protocol that can be executed by major LLMs and tailored or edited by employers, team leads, or educators.

The protocol focuses on:

- interaction states
- transitions and checkpoints
- gated tutoring flow
- diagnostic signals
- response constraints
- reducing premature answer disclosure
- stabilizing tutoring behavior across longer sessions

Published with Ethan Jordan.  
DOI: https://doi.org/10.54254/2755-2721/2026.CH32181

A public repository with protocol examples, sanitized tutor modules, templates, and case studies is being organized.

### TCG Verify — Computer Vision Authentication System

Original creator and technical lead for **TCG Verify**, an OpenCV + PyTorch computer vision project for authentic-vs-counterfeit trading card classification from smartphone images.

The project includes:

- card detection and perspective alignment
- native forensic ROI extraction
- PyTorch CNN modeling
- leakage-aware grouped validation/testing
- model failure analysis and stress testing
- local desktop app integration with PySide6/Qt
- Windows/macOS executable prototype builds

TCG Verify is currently being handled carefully while publication strategy is finalized. Full source code, trained weights, dataset artifacts, executable builds, and manuscript materials are not public at this time.

### Neural Networks from First Principles

Author of a free, code-backed educational guide on deep learning foundations. The guide is approximately 300 pages and is designed around a teaching structure of:

> plain English → conceptual math → code implementation → plain-English interpretation

Topics include preprocessing, tensors, NumPy, PyTorch workflows, MLPs, CNNs, RNNs, and foundational neural-network reasoning. The guide was originally created as a tutoring tool, but after strong positive feedback, it is being polished for a free public release.
